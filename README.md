# Shell Scripting Using Bash

This repository contains Bash shell scripts created for learning and practicing shell scripting on Linux systems.

## Objectives
- Understand Bash scripting fundamentals
- Automate repetitive tasks
- Work with Linux commands, files, and processes
- Improve command-line efficiency

## Topics Covered
- Basic shell commands
- Variables and user input
- Conditional statements (if-else, case)
- Loops (for, while)
- Functions
- File and directory handling
- Script automation
- Permissions and execution

## Repository Structure
basic-scripts/
automation-scripts/
file-handling/

## How to Run Scripts
Make the script executable and run it:

vim script_name.sh
chmod 777 script_name.sh
./script_name.sh
